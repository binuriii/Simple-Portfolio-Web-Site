# Responsive Portfolio Website
A multi-page portfolio website built using HTML5 and CSS3.
The project demonstrates semantic HTML structure, modern CSS styling techniques, and responsive design using media queries.

## Features
- Semantic HTML5 page structure
- Multi-page website
- Responsive layout for mobile and desktop
- CSS3 styling using a single external stylesheet
- Advanced CSS selectors and pseudo classes
- Transitions, transforms, and animations
- Form with HTML5 validation
- Clean, well-structured and commented code

## Pages Included
### Main Page
- Semantic HTML5 layout
- Sidebar section with additional links or social media
### About Page
- Personal information section
- Structured using semantic elements
### Blog / Projects Page
- Contains two blog posts
- Uses the <time> element with the datetime attribute
### Contact Page
- HTML5 form for:
- Newsletter registration
- Sending messages
- Uses modern input types and validation

## CSS Features Used
- The stylesheet includes a variety of CSS3 features such as:
- Custom fonts (Google Fonts / @font-face)
- Attribute selectors
- Pseudo elements
- User action pseudo classes
- Structural selectors
- :nth selectors
- Negation pseudo class
- Validity pseudo class
- Relational selectors
- Borders and rounded corners
- Gradients
- Box shadows
- Text effects
- Transforms
- Transitions
- Animations

## Responsive Design
- The website is fully responsive and adapts to different screen sizes using CSS media queries, ensuring good usability on:
- Desktop
- Tablet
- Mobile devices
